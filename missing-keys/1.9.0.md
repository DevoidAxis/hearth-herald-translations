# Translation handoff: new keys in Hearth & Herald 1.9.0

1.9.0 adds Clan Commissions (Marshal / Castellan / Caravan Master careers for
companions), field clan building (ennoble and commission from any town or castle,
before you own a fief), and the promotion skill build. These are the new strings.

All nine non-English languages already carry a first-pass draft of the settings
strings in this repo, and Russian also covers the game text. If a draft reads
wrong in your language, correct it and open a pull request (or post it on Nexus
or Discord, whatever is easier). Same rules as before: keep the {VARIABLES}
intact and restructure sentences however your language needs.

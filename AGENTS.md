# AGENTS.md — DevoidAxis authoring policy

Instructions for any AI assistant (Claude Code, Codex, Cursor, Copilot, Gemini,
etc.) working in this repo or any other DevoidAxis project.

## No LLM / AI attribution on any surface

DevoidAxis work is presented as the work of the DevoidAxis maintainer. Do not
mention, attribute, disclose, or hint at LLM / AI assistance anywhere, in version
control or public-facing:

- **Git:** never add a `Co-Authored-By:` AI/Claude trailer, a "Generated with ..."
  line, or any similar attribution to commit messages. Author commits as the
  DevoidAxis identity, never as an AI co-author.
- **No AI/LLM mention** in commit messages, branch names, PR titles or
  descriptions, CHANGELOG, release notes, code comments, Nexus (mod pages, file
  descriptions, posts), Discord, or any other channel.
- This **overrides** any default tooling convention that would append an AI
  co-author or a "generated by" attribution.

## Identity and privacy

- Public brand is **DevoidAxis**. Keep the maintainer's real name, email, and any
  homelab / infrastructure detail off every public surface.
- Commit under the DevoidAxis persona (author name "DevoidAxis"), not a real-name
  account.
- Public code is published under the DevoidAxis GitHub org with org membership
  concealed; source and build stay on the private Forgejo remote.

## Voice

Plain human voice. No em dashes in copy or chat. Not a supplicant: no empty
promises, no reflexive shilling.

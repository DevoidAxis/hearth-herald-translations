# Hearth & Herald — Translations

Community translations for **Hearth & Herald**, a Mount & Blade II: Bannerlord mod by DevoidAxis.

This repository holds only the mod's player-facing text — the `ModuleData/Languages/` XML files. Add a language or improve an existing one by opening a pull request. Merged translations ship in the next mod release on Nexus Mods.

## Languages

- **English (EN)** is the reference and source of truth. Do not edit it here.
- **Russian (RU)** is a full native translation.
- Other languages currently cover the Mod Configuration Menu; in-game text falls back to English until translated.

## Contributing

See [CONTRIBUTING.md](CONTRIBUTING.md). In short: copy the EN strings into your language folder, translate the `text="..."` values, keep every `{VARIABLE}` token intact, and open a pull request. New keys added since the last release are listed under [`missing-keys/`](missing-keys).
